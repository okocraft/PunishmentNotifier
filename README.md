# PunishmentNotifier

AdvancedBan での処罰時に Discord にログを送る BungeeCord プラグイン

## Requirements

- Java 11+
- BungeeCord
- AdvancedBan

## Usage

- サーバーディレクトリの `/plugins/` に配置し、サーバーを再起動する。

- 作成された `config.yml` にて `discord-webhook-url` に Webhook の URL を設定する。

- `/pnreload`
